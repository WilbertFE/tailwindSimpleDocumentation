Apa itu TailwindCSS?

tailwindcss adalah framework css yang menganut prinsip utility-first yang diimplementasikan dalam ratusan-ribuan nama nama kelas utility.

Mengapa Tailwind?

karena tailwind css memungkinkan anda untuk bekerja dengan html dan css sekaligus, sehingga tidak membuat pusing, dan juga tidak perlu membuat file css yang banyak. berikut alasannya,

  • tidak usah memikirkan nama kelas
  • tidak usah pindah-pindah file
  • mengatur responsive dengan mudah
